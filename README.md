# Zeni

シンプルにお金の管理をするための家計簿アプリです。
既存の家計簿アプリは次のような点を解結します。

- データをダウンロードするには有料会員でなければ出来ない
- ダウンロード出来る内容が不足している
- 高機能すぎて使いにくい

# Feature

- シンプルな入力画面と統計画面
- 自分の全てのデータをダウンロード可能
